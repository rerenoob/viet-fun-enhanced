# Vietnamese Learning App for Alan (Mỡ) - Enhanced Edition 🇻🇳

Enhanced Vietnamese learning app with pronunciation features for Alan (Mỡ).

## Live Demo
https://rerenoob.github.io/viet-fun-enhanced/

## Features

### 🎯 Pronunciation Features
- **🔊 Tap-to-Speak** - Tap any Vietnamese word to hear it pronounced
- **🎵 Tone Explorer** - Learn 6 Vietnamese tones with audio examples
- **🔧 Voice Diagnostics** - Check available voices and troubleshoot pronunciation
- **🤖 Roomba Commands** - Tap Roomba commands to hear pronunciation

### 🎮 Games
1. **🃏 Flashcards** - Learn vocabulary with tap-to-flip cards
2. **🔤 Word Match** - Match English words with Vietnamese translations
3. **✏️ Fill the Blank** - Complete Vietnamese sentences
4. **🎵 Tone Explorer** - Learn Vietnamese tones with audio
5. **🤖 Roomba Says!** - Follow Roomba's commands in Vietnamese

### 🚀 Technical Features
- **Web Speech API** integration with Vietnamese voice detection
- **iOS Safari compatibility** with user gesture handling
- **Voice loading retries** for better reliability
- **Diagnostic console function** `checkVoices()` for troubleshooting

## How to Use

1. **Tap any Vietnamese word** to hear it pronounced
2. **Explore tones** in the Tone Explorer section
3. **Check pronunciation** if voices aren't working (Pronunciation Help screen)
4. **Play all games** to learn vocabulary and practice

## Pronunciation Setup

If Vietnamese pronunciation doesn't work:
1. Tap the **Pronunciation Help** button on the main menu
2. Run **Check Available Voices** to see what voices are available
3. Install Vietnamese language pack on your device if needed
